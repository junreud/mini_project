# 주식 투자 도움 챗봇 미니 프로젝트 📈

> 학습용 LLM Fine-tuning 프로젝트 - 주식 투자 기초 지식을 제공하는 한국어 챗봇

## 📌 프로젝트 목표
- 주식 투자 초보자를 위한 기초 지식 제공 챗봇
- QLoRA를 활용한 효율적 LLM Fine-tuning 경험
- 전체 파이프라인 구축 (데이터 → 학습 → 배포)

## 🤖 챗봇 특징
- **주제**: 주식 투자 기초 지식, 용어 설명, 투자 원칙
- **성격**: 친근하면서도 신중한 조언자
- **특징**: 투자 판단은 본인 책임임을 항상 강조
- **목적**: 교육 및 학습용 (투자 권유 아님)

## 🔄 현재 진행 단계
- [x] Phase 1: 환경 구성 ✅
- [x] Phase 2: 주식 투자 데이터 준비 ✅
- [ ] Phase 3: 모델 로딩 & 테스트
- [ ] Phase 4: Fine-tuning
- [ ] Phase 5: 평가 & 개선
- [ ] Phase 6: 배포

## 🛠️ 기술 스택
- **Base Model**: beomi/llama-2-ko-7b (예정)
- **Method**: QLoRA (4-bit quantization)
- **Data**: 주식 투자 관련 한국어 대화 데이터 (15개 쌍)
- **Format**: Llama-2 Chat Template

## � 데이터 구성
- **총 15개** 대화 쌍
- **주제 분류**:
  - 투자 기초 (시작 방법, 계좌 개설)
  - 투자 지표 (PER, PBR, 배당)
  - 투자 전략 (분산투자, 손절매)
  - 투자 상품 (ETF, 개별 주식)
  - 리스크 관리

## ⚠️ 면책 조항
이 챗봇은 **교육 및 학습 목적**으로만 사용됩니다.
- 제공하는 정보는 일반적인 투자 지식입니다
- 투자 권유나 종목 추천이 아닙니다
- 실제 투자 결정과 그에 따른 손익은 본인의 책임입니다
